# Parcel Starter Template

## Development

Clone this repo and cd into it

```
npm i
npm start
```

## Production

```
npm run build
```

## Features

Supports

- `Autoprefixr`
- `SASS`
- `typescript`
- `babel` for transpiling es6 code

## Env Vars

- Edit .env file and access it as follows

```
process.env.ENV_VAR_NAME
```
